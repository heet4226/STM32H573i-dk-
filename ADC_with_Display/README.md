This project is implemented using the STM32H573I-DK development board. The board includes an internal temperature sensor, whose output is measured using the on-chip Analog-to-Digital Converter (ADC). The raw ADC data is processed and converted into a meaningful temperature value. Both the ADC reading and the calculated temperature are displayed in real time on the boardâ€™s on-board TFT display.
